# Homebrew Tap for NCP Nuke

This repository contains the Homebrew formulae for [ncp-nuke](https://github.com/YOUR_GITHUB_USERNAME/ncp-nuke).

## Usage

```bash
brew tap YOUR_GITHUB_USERNAME/tap
brew install ncp-nuke
```

## Maintenance

This repository is automatically updated by [GoReleaser](https://goreleaser.com) when a new version of `ncp-nuke` is released.
